# Feature Roadmap

## Phase 0 (Project skeleton)
- Project structure
- Config loading
- CLI: info + health

## Phase 1 (RTL-SDR core wins)
1) Band Scanner -> CSV export
2) Noise Floor Profiler -> CSV export
3) A/B Antenna Comparator -> score + CSV
4) Waterfall Recorder -> file export (later)
5) Frequency bookmarks + notes
6) Alerts on frequencies (simple threshold)
7) Run report (JSON) for automation/n8n

## Phase 2 (Reporting + polish)
- Plots (matplotlib)
- Session-based “report pack” export
- Better calibration/baseline normalization

## Phase 3 (More instruments)
- NanoVNA plugin (S11/SWR)
- TinySA plugin
- Optional SoapySDR support to unify devices

## Phase 4 (GUI optional)
- Simple desktop UI (later)
